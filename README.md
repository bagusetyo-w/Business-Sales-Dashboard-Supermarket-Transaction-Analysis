# Business-Sales-Dashboard-Supermarket-Transaction-Analysis
Project akademik ini bertujuan untuk menganalisis performa penjualan supermarket berdasarkan data transaksi, cabang, metode pembayaran, produk, dan rating pelanggan. Analisis dilakukan menggunakan Microsoft Excel dan divisualisasikan dalam dashboard interaktif menggunakan Power BI untuk mendukung pengambilan keputusan bisnis.
